## Suggested Next Steps

After completing this tutorial, useful next projects include:

1. Run a longer BCI simulation.
2. Compare inventory initialization with bare-ground initialization.
3. Create a custom two-PFT parameter file.
4. Modify trait values and explore coexistence.
5. Run a simulation for a new site.
6. Generate custom meteorological forcing.
7. Create a regional simulation.
8. Analyse FATES outputs in Python or R.
9. Compare simulations against field observations.
10. Build your own FATES workflow for a research question.
